## 启用 WebNN

> <InfoIcon /> Windows ML / OpenVINO 执行提供需要 Windows 11 版本 24H2 或更高版本才能运行的程序

<Steps>
### 安装最新的 [Chrome Canary](https://google.com/chrome/canary) 或 [Edge Canary](https://www.microsoft.com/en-us/edge/download/insider?form=MA13FJ)
### 导航到浏览器地址地址的`about://flags`
### 搜索 `Enables WebNN API` 将其更改为 `Enabled`
###重新启动浏览器
</Steps>

## 在 LiteRT 控制台（CPU 和 GPU）上运行 WebNN

> <InfoIcon /> 24H2 Windows 11 版本之前、Windows 10 和 Linux

<Steps>
### 导航到浏览器地址地址的`about://flags`
### 搜索 `ONNX Runtime backend for WebNN` 将其更改为 `Disabled`
### 搜索 `DirectML backend for WebNN` 将其更改为 `Disabled`
###重新启动浏览器
</Steps>
