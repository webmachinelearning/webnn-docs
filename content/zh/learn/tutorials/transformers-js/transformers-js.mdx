[Transformers.js](https://huggingface.co/docs/transformers.js/en/index) 提供基于浏览器的机器学习功能，镜像 Hugging Face Transformers Python 库。它支持跨多个领域的各种预训练模型。

在底层，[Transformers.js](https://huggingface.co/docs/transformers.js/en/index) 使用 ONNX Runtime 在浏览器中执行模型。您可以使用 Optimum 库将现有 PyTorch、TensorFlow 或 JAX 模型转换为 ONNX 格式。

```js showLineNumbers filename="example.js"
import * as transformers from from '@huggingface/transformers';

const path = 'xenova/resnet-50';
const options = {
  dtype: 'fp16',
  device: 'webnn-gpu', // 'webnn', 'webnn-cpu', 'webnn-npu'
  session_options: {
      freeDimensionOverrides: {
        batch_size: 1,
        num_channels: 3,
        height: 224,
        width: 224,
      },
      context: {},
  },
};

const classifier = await transformers.pipeline('image-classification', path, options);
let [err, output] = await asyncErrorHandling(classifier(imageUrl, { topk: 3 }));
```

> <InfoIcon /> WebNN 目前不支持动态形状，需要设置 [`free_dimension_overrides`](../transformers-js/free-dimension-overrides)。

> <InfoIcon /> [`free_dimension_overrides`](../transformers-js/free-dimension-overrides) 可以在上面的示例代码中设置，也可以在模型的 config.json 中设置为“transformers.js_config”中的字段。
> 如果未设置 `free_dimension_overrides`，您可能会遇到性能显着下降的情况。
