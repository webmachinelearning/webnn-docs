## 自由尺寸覆盖

ONNX 模型支持通过自由维度动态输入形状。自由维度允许模型接受沿该维度的不同大小的输入。例如，输入形状为 `[batch, 3, height, width]` 的模型可以处理任意数量、任意尺寸的 RGB 图像（3 个通道）。

虽然灵活，但当您的应用程序使用一致的输入大小时，您可以通过固定这些尺寸来优化性能。例如，如果您的 WebNN 应用程序始终处理单个 `224×224` 图像，您可以使用 `freeDimensionOverrides` 选项覆盖从 `[batch, 3, height, width]` 到 `[1, 3, 224, 224]` 的自由尺寸。此配置将输入维度锁定为特定值，从而可能提高模型性能。

### 在 Transformers.js 3.5.0+ 中设置自由维度覆盖

在 [Transformers.js](https://huggingface.co/docs/transformers.js/index) 3.5.0 或更高版本中使用动态形状时，您可以使用以下方法之一指定自由尺寸覆盖（按优先顺序列出）：

1. 直接在代码中使用 `freeDimensionOverrides`
2. 在模型的 `config.js` 文件内 `transformers.js_config` 的 `device_config` 级别
3. 在模型的 `config.js` 文件内的 `transformers.js_config` 模型级别中

将使用第一个提供有效覆盖的方法，较早的方法优先于较晚的方法。

#### 优先级1

使用 Transformers.js 在代码中设置 `freeDimensionOverrides`。

```js showLineNumbers filename="*.js"
import * as transformers from from '@huggingface/transformers@3.5.0';

const options = {
  device: "webnn-gpu", // webnn, webnn-cpu, webnn-npu
  dtype: "fp32",
  session_options: {
    freeDimensionOverrides: {
      batch: 1,
      height: 224,
      width: 224
    }
  }
}

const classifier = await transformers.pipeline('image-classification', path, options);
```

#### 优先级2

在 Huggingface 模型的 `config.js` 文件中的 `transformers.js_config` 的 `device_config` 级别设置 `free_dimension_overrides`，例如[Xenova/mobileclip_s0](https://huggingface.co/Xenova/mobileclip_s0/blob/main/config.json)

```js showLineNumbers filename="config.json"
{
  "transformers.js_config": {
    "device_config": {
      "webnn-gpu": {
        "dtype": "fp16",
        "free_dimension_overrides": {
          "batch": 1,
          "height": 224,
          "width": 224
        }
      }
    }
  }
}
```

#### 优先级3

在 Huggingface 模型的 `config.js` 文件中的 `transformers.js_config` 模型级别中设置 `free_dimension_overrides`，例如[Xenova/mobileclip_s0](https://huggingface.co/Xenova/mobileclip_s0/blob/main/config.json)

```js showLineNumbers filename="config.json"
{
  "transformers.js_config": {
    "device": "webnn-gpu",
    "dtype": "fp16",
    "free_dimension_overrides": {
      "batch": 1,
      "height": 224,
      "width": 224
    }
  }
}
```
