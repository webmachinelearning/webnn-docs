# JavaScript 机器学习框架

WebNN API 正在集成到主要的 JavaScript 机器学习框架中，例如 ONNX Runtime Web、Transformers.js 和 LiteRT 。这使得 Web 开发人员能够使用这些高级框架以更少的代码构建 AI 应用程序，同时利用 WebNN 的、低级神经网络操作。

# 普通 JavaScript

直接使用WebNNAPI创建自定义计算图和神经网络模型，在您的AI应用程序中具有更大的灵活性。但是，与使用ONNX和TFLite等成熟解决方案相比，使用原始WebNNAPI构建模型需要更复杂的代码。

直接使用 WebNN JavaScript API 代替 ML 框架，通过消除框架劳动力来减少应用程序的大小。
