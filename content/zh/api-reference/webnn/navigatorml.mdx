# 导航器.ml

`navigator.ml` 属性提供对 WebNN API 的访问，允许 Web 应用程序与浏览器中的机器学习功能进行交互。

## 网络IDL

```idl showLineNumbers filename="web-idl"
interface mixin NavigatorML {
  [SecureContext, SameObject] readonly attribute ML ml;
};
Navigator includes NavigatorML;
WorkerNavigator includes NavigatorML;
```

## 语法

```js showLineNumbers filename="navigator.ml.js"
const ml = navigator.ml
```

＃＃＃价值

提供对 WebNN 功能的访问 [`ML`](https://www.w3.org/TR/webnn/#api-navigator-ml) 对象。

＃＃描述

`navigator.ml` 属性返回对 `ML` 对象的注释，该对象充当基于浏览器的机器学习操作的入口点。接口是：

- 窗口和专用工作环境均可用
- 可通过窗口上下文中的`Navigator`界面进行访问
- 可通过专用工作环境中的 `WorkerNavigator` 界面进行访问
- 受安全上下文要求（HTTPS 或本地主机）保护

＃＃例子

```js showLineNumbers filename="example.js"
if ('ml' in navigator) {
  // ML API is supported
  const mlInstance = navigator.ml;
  // Use ML functionality here
} else {
  console.log('ML API is not supported in this browser');
}
```
＃＃规格

| WebNN API | Status |
|--------------|---------|
| [`navigator.ml` interface](https://www.w3.org/TR/webnn/#api-navigator-ml) | Candidate Recommendation Draft |

## 安全要求

访问 `navigator.ml` 需要安全上下文（HTTPS 或本地主机）。尝试在非安全上下文中访问此 API 将导致错误。

## 浏览器兼容性

> <InfoIcon /> `navigator.ml`界面是实验性的，不同浏览器的支持可能会有所不同。

参见[Browser Compatibility: WebNN API](browser-compatibility#webnn-apis)
