# MLTensor

`MLTensor`接口表示可用于`MLGraph`的输入或输出的张量。

## MLTensorDescriptor

在深入研究`MLTensor`之前，让我们先了解一下描述其特性的`MLTensorDescriptor`字典。

### 网络IDL

```webidl showLineNumbers filename="web-idl"
dictionary MLTensorDescriptor : MLOperandDescriptor {
  boolean readable = false;
  boolean writable = false;
};
```

＃＃＃特性

- `readable` （布尔值）
- 确定是否可以使用 `readTensor(tensor)` 或 `readTensor(tensor, outputData)` 读取张量的内容
- 默认值： `false`

- `writable` （布尔值）
- 确定是否可以使用 `writeTensor()` 写入张量的内容
- 默认值： `false`

## MLTensor 接口

### 网络IDL

```webidl showLineNumbers filename="web-idl"
[SecureContext, Exposed=(Window, DedicatedWorker)]
interface MLTensor {
  readonly attribute MLOperandDataType dataType;
  readonly attribute FrozenArray<unsigned long> shape;
  readonly attribute boolean readable;
  readonly attribute boolean writable;
  undefined destroy();
}
```

＃＃＃特性

- `dataType` （论）
- 返回其转发中指定的张量的数据类型
- 类型： `MLOperandDataType`

- `shape` （论）
- 返回其请求中指定的张量维度
- 类型： `FrozenArray<unsigned long>`

- `readable` （论）
- 返回张量的内容是否可以读取
- 类型： `boolean`

- `writable` （论）
- 返回张量的内容是否可写
- 类型： `boolean`

＃＃＃方法

＃＃＃＃破坏（）

释放与 MLTensor 关联的资源。

##### 语法

```javascript
tensor.destroy()
```

#####返回值

- `undefined`

＃＃＃＃＃描述

`destroy()`方法是幂等的，这意味着在同一个上多次调用此方法将与单次调用具有相同的效果。

### 内存分配

支持 `MLTensor` 的内存根据相关 `MLContext` 和 `MLTensorDescriptor` 的实现定义要求进行分配。这些要求可能包括：

- 十个特定项目合同
- 内存特定池的分配
- 其他具体实施要求

## 示例

###创建和使用MLTensor

```js showLineNumbers filename="example-1.js"
 async function mlTensor() {
  // Define tensor descriptor
  const tensorDescriptor = {
    dataType: "float32",
    shape: [1, 28, 28, 1],
    readable: true,
    writable: true
  };
 
  try {
    // Create tensor through MLContext and MLTensorDescriptor
    const context = await navigator.ml.createContext({
      deviceType: 'gpu'
    });
    const tensor = await context.createTensor(tensorDescriptor);

    // Use the tensor
    console.log("Tensor shape:", tensor.shape);
    console.log("Data type:", tensor.dataType);
    
    // Clean up when done
    tensor.destroy();
  } catch (error) {
    console.error("Error creating tensor:", error);
  }
}
```

### 读取MLTensor数据

```js showLineNumbers filename="example-2.js"
async function readTensorData(tensor) {
  if (tensor.readable) {
    try {
      const data = await context.readTensor(tensor);
      console.log("Tensor data:", data);
    } catch (error) {
      console.error("Error reading tensor:", error);
    }
  }
}
```
