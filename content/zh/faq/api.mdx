# API

## WebNN API 的编程模型是什么？

WebNN（Web 神经网络 API）为在 Web 浏览器中执行机器学习模型提供了底层接口。

它专门专注于神经网络操作和执行基础设施，将模型加载和解析等更高级别的任务委托给专门的库（例如 ONNX Runtime Web、LiteRT）或 Web 应用程序本身。这种设计使得 WebNN 与格式无关，并保持了清晰的关注点分离。

它的编程模型包括两个主要步骤：模型构建和执行。

### 核心步骤

1. 模型构建
   `MLGraphBuilder` API 构建表示您的模型的计算图。一旦定义，此图将被编译为可执行形式。

2. 模型执行
   编译后的图处理输入数据以生成预测或分类。WebNN 提供了显式或根据功能需求选择合适的 ML 后端的方法。

### 硬件加速

WebNN 自动利用可用的硬件加速器（CPU、GPU、NPU、TPU）来优化模型执行。这种与硬件无关的方法确保了：
- 跨不同设备的最佳性能
- 降低推理延迟
- 流畅的用户体验
