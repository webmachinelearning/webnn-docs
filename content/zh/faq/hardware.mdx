
# 硬件和 NPU

## WebNN可以在NPU上运行吗？

是的，WebNN API 旨在支持神经处理单元 (NPU)。API 可以在从台式电脑到移动平台的各种设备上使用 NPU。

已验证 WebNN 与以下设备兼容：

- 英特尔酷睿超级处理器
- 流星湖 (MTL)

- 英特尔酷睿 Ultra 系列 2 处理器
- 月湖 (LNL)
- 箭湖 (ARL)

- 英特尔酷睿 Ultra 系列 3 处理器
- 黑豹湖 (PTL)

## 如何在浏览器中监控 WebNN 可用的硬件资源？

您可以使用多种浏览器和系统工具监控系统资源：

- 有关 GPU 信息：

- 访问 about://gpu 查看详细的 GPU 诊断和错误日志
- 检查 about://histograms/webnn 的 WebNN - 特定性能指标

- 对于一般系统资源：

- 浏览器资源：打开Google Chrome或Microsoft Edge任务管理器（菜单→更多工具→任务管理器）以监控内存使用情况和CPU消耗
- 系统资源：使用Windows任务管理器监控CPU、GPU和NPU利用率
