## 如何向 WebNN 提交问题（Issue）？

将问题提交到以下 WebNN 问题跟踪器：

- [设计或规范](https://github.com/webmachinelearning/webnn/issues)
- [Chromium 实现 (组件: Chromium > Blink > WebML)](https://issues.chromium.org/issues?q=status:open%20componentid:1456206&s=created_time:desc)
- [ONNX Runtime Web 或 WebNN Execution Provider](https://github.com/microsoft/onnxruntime/issues)
- [WebNN 示例](https://github.com/webmachinelearning/webnn-samples/issues)
- [WebNN 开发者预览演示](https://github.com/microsoft/webnn-developer-preview/issues)

如果您不确定选择哪种问题类型，请随时在[这里](https://github.com/webmachinelearning/webnn-samples/issues)提交您的问题，我们将帮助分类和解决。

## 如何调试 WebNN 问题？

WebNN API 使用标准的 DOM 异常来传播错误。当发生错误时，API 会抛出特定的异常，您可以在代码中捕获并处理这些异常。

查看 GPU 信息日志，这些页面包含有关 GPU 相关错误、直方图和性能的详细诊断信息。

```javascript
about://gpu
about://histograms/webnn
```

- 请查阅 [W3C WebNN 规范](https://www.w3.org/TR/webnn/) 获取完整的错误处理文档。
- 欢迎在[这里](https://github.com/webmachinelearning/webnn-samples/issues)提出您的问题，我们将帮助分析并尝试解决。
